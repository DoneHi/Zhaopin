# Zhaopin
爬取西安电子科技大学就业信息网+生成图片+设置定时任务，每天6，12，18点自动发送最新的就业信息，以及分析发送到邮箱里

## l.py将爬取下来的信息画图

## em/p.py 发送超链接的表格的信息，也就是把爬取下的信息，进行表格化发送给邮箱

## em/w.py 将生成的图片发送给邮箱用户里
